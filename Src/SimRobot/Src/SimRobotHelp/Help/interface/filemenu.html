File Menu

The File menu offers the following commands:

  • New
    Creates a new simulation.

    Use this command to create a new simulation in SimRobot. This command closes the current simulation.
    You can open an existing simulation with the Open command.
    Tool Bar:
    Keys: CTRL+N
  • Open
    Opens an existing simulation.

    Use this command to open an existing simulation. This command closes the current simulation. If you have opened the simulation you can open one of the last four simulations with the 1, 2, 3, 4 command.
    You can create new simulations with the New command.
    Tool Bar:
    Keys: CTRL+O
  • Close
    Closes the current simulation.

    Use this command to close the current simulation. SimRobot suggests that you save changes to your simulation before you close it. If you close a simulation without saving, you lose all changes made since the last time you saved or you compiled it. Before closing an untitled simulation, SimRobot displays the Save As dialog box and suggests that you name and save the simulation.
  • Save
    Saves an opened simulation using the same file name.

    Use this command to save the current simulation to its current name and directory. When you save a simualtion for the first time, SimRobot displays the Save As dialog box so you can name your simualtion. If you want to change the name and directory of an existing simulation before you save it, choose the Save As command.
    Tool Bar:
    Keys: CTRL+S
  • Save as
    Saves an opened simulation to a specified file name.

    Use this command to save and name the current simulation. SimRobot displays the Save As dialog box so you can name your simulation.
    To save a simulation with its existing name and directory, use the Save command.
  • 1, 2, 3, 4
    Opens one of the last four simulations.

    Use the numbers and filenames listed at the bottom of the File menu to open one of the last four simulations you closed. Choose the number that corresponds with the simulation you want to open.
  • Exit
    Exits SimRobot.

    Use this command to end your SimRobot session. You can also use the Close command on the SimRobot Control menu. SimRobot prompts you to save the current simulation if it contains unsaved changes.
    Window:
    Keys: Alt+F4
