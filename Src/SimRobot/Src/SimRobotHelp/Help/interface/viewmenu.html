View Menu

The View menu offers the following commands:

  • Toolbar
    Shows or hides the toolbar.

    Use this command to display and hide the Toolbar, which includes buttons for some of the most common commands in SimRobot. A check mark appears next to the menu item when the Toolbar is displayed.
    See Toolbar for help on using the Toolbar.
  • Status Bar
    Shows or hides the status bar.

    Use this command to display and hide the Status Bar, which describes the action to be executed by the selected menu item or depressed toolbar button, and keyboard latch state. A check mark appears next to the menu item when the Status Bar is displayed.
    See Status Bar for help on using the Status Bar.
  • Console
    Shows or hides the console window.

    Use this command to display and hide the Console window. A check mark appears next to the menu item when the Console window is displayed.
    See Console for help on using the Console.
  • Tree Window
    Shows the tree window.

    Use this command to open or activate the Tree window.
    This command is unavailable if the Tree Window is focussed.
    Tool Bar:
    Keys: CTRL+T
  • Camera
    This command is unavailable if the Object Window is focussed.

    Camera Modes:

    • Target Mode
    • Free Mode
    • Presentation Mode

    Camera Commands:

    • Reset
      Keys: R
    • Toggle
      Keys: T
    • Fit
      Keys: F
    • SnapToRobot


  • Vertical Opening Angle
    This command is unavailable if the Object Window is focussed.
  • Surface Rendering
    This command is unavailable if the Object Window is focussed.

    There are several options available:

    • Appearance Wire Frame
    • Appearance Flat Shading
    • Appearance Smooth Shading
    • Appearance Texture Shading
    • Appearance Camera Simulation
    • Physics Wire Frame
    • Physics Flat Shading
    • Physics Smooth Shading
    • Mixed Mode 1 (Shaded Physics)
    • Mixed Mode 2 (Shaded Appearance)

  • Show Controller Drawings
    This command is unavailable if the Object Window is focussed.
  • Show Sensors
    This command is unavailable if the Object Window is focussed.

    Toolbar:
    Keys: S
  • Selected Item
    This command is unavailable if the Tree Window is focussed.
  • Line Graph
    This command is unavailable if the Sensor Window is focussed.

    Toolbar:
    Keys: L
  • Column Graph
    This command is unavailable if the Sensor Window is focussed.

    Toolbar:
    Keys: O
  • Monochrome Image
    This command is unavailable if the Sensor Window is focussed.

    Toolbar:
    Keys: M
  • Color Image
    This command is unavailable if the Sensor Window is focussed.

    Toolbar:
    Keys: C
  • Stereogram
    This command is unavailable if the Sensor Window is focussed.

    Toolbar:
    Keys: S
  • Grid
    This command is unavailable if the Sensor Window is focussed.

    Toolbar:
    Keys: G
  • Brightening
    This command is unavailable if the Sensor Window is focussed.
