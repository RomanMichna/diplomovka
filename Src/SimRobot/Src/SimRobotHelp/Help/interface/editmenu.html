Edit Menu

The Edit menu offers the following commands:

  • Undo
    Resets the last change you have made.

    Use this command to undo the last change you have made. If you undo twice, you undo your undo and nothing will have changed.
    This command is only available if an Editor Window is focussed.
    Keys: CTRL+Z
  • Cut
    Cuts selected text into the clipboard.

    Use this command to remove the currently selected text from the editor and put it on the clipboard. This command is unavailable if there is no text currently selected.
    Cutting text to the clipboard replaces the contents previously stored there.
    Tool Bar:
    Keys: CTRL+X
  • Copy
    Copies selected text into the clipboard.

    Use this command to copy the selected text, object or sensor data onto the clipboard. This command is unavailable if there is no text, no object or sensor currently selected. You can also use this command to copy the actual view onto the clipboard.
    Copying data to the clipboard replaces the contents previously stored there.
    Tool Bar:
    Keys: CTRL+C
  • Paste
    Pastes the contents from the clipboard.

    Use this command to insert a copy of text on the clipboard at the insertion point. This command is unavailable if the clipboard is empty.
    This command is only available if an Editor Window is focussed.
    Tool Bar:
    Keys: CTRL+V
  • Indent
    Indents selected text by one column.

    Use this command to indent the selected text by one column. If no text is selected the command inserts spaces up to the next tab position.
    This command is only available if an Editor Window is focussed.
    Keys: Tab
  • Unindent
    Unindents selected text by one column.

    Use this command to unindent the selected text by one column. This command is unavailable if there is no text currently selected.
    This command is only available if an Editor Window is focussed.
    Keys: Shift+Tab
  • Find
    Finds text you can enter.

    Use this command to search for a specific text. The command displays the Find dialog box, that enables you to specify the search text and some options.
    Use the Find Next command to search for the text's next occurence in the simulation after you closed the Find dialog box.
    Use the Replace command to replace the found text by a different text.
    This command is only available if an Editor Window is focussed.
    Keys: Ctrl+F
  • Find Next
    Finds the next occurrence of the text you are looking for.

    Use this command to search for the next occurence of a text you specified in the last Find command.
    This command is only available if an Editor Window is focussed.
    Keys: F3
  • Replace
    Finds text you can enter and replaces it with other text you can enter, too.

    Use this command to search for a specific text and replace it by a different text. The command displays the Replace dialog box, that enables you to specify the search and replace texts and some options.
    Use the Find command if you only want to search for a text without replacing it by another.
    This command is only available if an Editor Window is focussed.
    Keys: Ctrl+H
  • Drag and Drop Plane
    Lets to drag and drop plane

    This command is only available if an Object Window is focussed.

    There are three options:
    Mode Shortcut Description
    X/Y Plane Z HELP
    X/Z Plane Y HELP
    Y/Z Plane X HELP

  • Drag and Drop Mode
    Switches to Drag and Drop mode.

    This command is only available if an Object Window is focussed.

    There are three options:
    Mode Shortcut Description
    Keep Dynamics 8 HELP
    Reset Dynamics 9 HELP
    Apply Dynamics 0 HELP

This menu is not available if the Tree Window is focussed.
