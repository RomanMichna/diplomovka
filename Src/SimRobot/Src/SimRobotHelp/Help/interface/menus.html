Menus

The Simrobot Window Menus offer the functionalities you need to create and manipulate your own simulation. These menus are available:

  • File Menu Offers commands for File management
  • Edit Menu Offers commands for editing in the current window focussed
  • View Menu Offers commands on what you see
  • Simulation Menu Offers commands to control the simuation
  • Window Menu Offers commands on how the windows are arranged
  • Help Menu Offers command to display help window and about dialog
