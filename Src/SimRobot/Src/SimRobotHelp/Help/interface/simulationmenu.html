Simulation Menu

The Simulation menu offers the following commands:

  • Reset
    Resets the simulation to the beginning.

    Use this command to reset the simulation to the beginning. The following does happen:
    - If you have defined a Controller for the current simulation, this Controller object is destructed.
    - The current SIMULATION object is destructed.
    - A new SIMULATION object is constructed and the simulation file is compiled.
    - If you have defined a Controller for the current simulation and if there are no compile errors, a new Controller object is constructed.
    - All views are updated.
    Tool Bar:
    Keys: Shift+F5
  • Start
    Starts or stops the simulation.

    Use this command to start or stop the simulation. If you have changed the simulation file the Reset command is executed first. After you started the simulation it runs until you use this command again or until the SetActors member of your Controller returns FALSE.
    You can see the number of the current simulation step in the Status Bar.
    Tool Bar:
    Keys: F5
  • Step
    Executes a single simulation step.

    Use this command the execute a single simulation step. If you have changed the simulation file the Reset command is executed first.
    You can see the number of the current simulation step in the Status Bar.
    Tool Bar:
    Keys: F8
