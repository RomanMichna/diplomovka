Help Menu

The Help menu offers the following commands, which provide you assistance with SimRobot:

  • Help Topics
    Offers you an index to topics on which you can get help.

    Use this command to display this help.
  • Help On Keyword
    Displays help for selected keyword.

    Use this command to display reference information about the selected keyword or object.
    Tool Bar:
    Keys: F1
  • Help On Object
    Displays help for selected object.

    Use this command to display reference information about the selected keyword or object.
    Tool Bar:
    Keys: F1
  • About Qt
    Displays information about Qt

    Use this command to display some information about Qt.
  • About SimRobot
    Displays information about SimRobot.

    Use this command to display some information about SimRobot.
