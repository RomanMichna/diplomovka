Window Menu

The Window menu offers the following commands, which enable you to arrange multiple windows in the SimRobot Window:

  • Cascade
    Arranges windows in an overlapped fashion.

    Use this command to arrange multiple opened windows in an overlapped fashion.
  • Tile Horizontally
    Arranges windows horizontally as non-overlapping tiles.

    Use this command to vertically arrange multiple opened windows in a non-overlapped fashion.
  • Tile Vertically
    Arranges windows vertically as non-overlapping tiles.

    Use this command to arrange multiple opened windows side by side.
  • Arrange Icons
    Arranges icons of closed windows.

    Use this command to arrange the icons for minimized windows at the bottom of the main window. If there is an open window at the bottom of the SimRobot window, then some or all of the icons may not be visible because they will be underneath this window.
  • 1, 2, ...
    Focusses the selected window.

    SimRobot displays a list of currently open windows at the bottom of the Window menu. A check mark appears in front of the name of the active window. Choose a window from this list to make it active.
